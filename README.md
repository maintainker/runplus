# 🏃 RunPlus - 러닝 트래킹 앱

<div align="center">
  <p>러닝/걷기 경로를 기록하고 건강 통계를 제공하는 모바일 앱</p>
</div>

## 📌 핵심 기능

| 기능               | 설명                             | 구현 상태 |
| ------------------ | -------------------------------- | --------- |
| 🛰️ 실시간 GPS 추적 | Google Maps 기반 운동 경로 기록  | ✅ 100%    |
| 📊 활동 통계       | 주간/월간 거리, 속도             | ⏳ 20%    |
| 🔐 계정 연동       | Supabase Auth로 다중 기기 동기화 | ✅ 100%    |
| 📱 오프라인 지원   | SQLite 로컬 저장 후 자동 동기화  | ❌ 0%     |
